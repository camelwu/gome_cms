Aeromind1.3.0 for Windows 我们正式更名为“Aeromind”啦！

发布日期：2017-10-10      发布版本：v1.3.0

1.我们正式更名为“Aeromind”啦，Aeromind给工作飞一般的感觉；
2.增加系统托盘消息盒子功能，快速预览单聊、群聊、频道未读新消息；
3.桌面端【设置】菜单增加【快捷按键】功能，支持【发送消息】和【换行】快捷键设置，【屏幕截图】和【打开窗口】自定义快捷键设置，检测热键冲突等功能，Windows支持【Esc】关闭窗口功能；
4.桌面端【更多】菜单增加【下载手机版】入口，可快速扫码下载移动端应用；
5.发送图片为原图，无任何压缩。
